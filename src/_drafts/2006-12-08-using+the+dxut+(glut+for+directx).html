Rather nice so long as your not putting together anything full-blown.

In any case, it seems a bit strange to setup:



A minimalist program to test your setup would look like the following:

#include "dxut.h"

int WINAPI
WinMain( HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nShowCmd )
{
DXUTInit();
DXUTCreateWindow();
DXUTCreateDevice();
DXUTMainLoop( NULL );

return DXUTGetExitCode();
}
